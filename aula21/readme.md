## Operadores de comparação

## >    : maior que
## >=   : maior que ou igual a
## <    : menor que
## <=   : menor que ou igual a
## ==   : igual a (valor) ========================== NÃO USAR
## ===  : igual a (valor e tipo)
## !=   : diferente de (valor) ========================== NÃO USAR
## !==  : diferente de (valor e tipo)
